# HR-Analytics-Dashboard
This project is an interactive HR Analytics Dashboard built using Streamlit. The goal was to analyze employee data and present it in a structured and meaningful way using visualizations.

Instead of keeping the analysis in a Jupyter Notebook, I wanted to build something more practical where users can upload a dataset and explore it themselves.

The dashboard is deployed online and can be accessed through the live link.

# What This Dashboard Includes

File upload option (CSV format)

Executive summary with key metrics

Attrition analysis by department

Age distribution analysis

Salary comparison charts

Correlation heatmap for numerical features

Interactive filters in the sidebar

Option to download filtered data

The idea was to simulate a real HR analytics scenario and present the insights in a clean dashboard format.

# Tech Stack

Python

Streamlit

Pandas

Plotly

GitHub

Streamlit Community Cloud

# How It Works

Upload an HR dataset in CSV format.

Use the filters to select department, gender, or attrition status.

The dashboard updates automatically.

Insights and visualizations change based on selected filters.

You can download the filtered dataset if needed.
